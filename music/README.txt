CARA DOWNLOAD & SETUP:

1. Download lagu dari YouTube:
   - Buka: https://www.youtube.com/watch?v=H7HmzwI67ec
   - Gunakan situs converter seperti:
     * https://ytmp3.nu/
     * https://ytmp3.cc/
     * https://y2mate.com/
   - Pilih format MP3 (128kbps atau 192kbps)

2. Download file MP3 yang sudah diconvert

3. Rename file menjadi: bg-music.mp3

4. Masukkan file ke folder ini (music/)

SPESIFIKASI FILE:
- Format: MP3 (paling kompatibel)
- Bitrate: 128kbps - 192kbps
- Ukuran: Maksimal 5MB agar loading cepat
- Durasi: ~3 menit (akan loop otomatis)
